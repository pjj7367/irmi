"첨부한 '민생위기 조기경보 레이더' 개발 계획서를 완벽히 분석해서 대시보드 UI를 새로 짜줘.

1. 디자인 컨셉: > - 전체적으로 애플(Apple)의 미니멀리즘과 토스(Toss)의 화이트/그레이 톤을 적용해줘.

배경은 #F9FAFB, 카드는 #FFFFFF로 하고, 폰트는 'Pretendard' 혹은 시스템 폰트를 사용해줘.

모든 요소의 모서리는 16px로 둥글게, 여백은 넓게 잡아서 전문적인 느낌을 줘.

2. 기능적 킥 (해커톤 포인트):

레이더 섹션: 화면 상단에 현재의 위기 단계를 보여주는 '레이더 차트'나 '게이지'를 넣어줘. (주의/경보/심각 단계에 따라 색상이 부드럽게 변하도록)

선행 지표 대조: 통계청 데이터(느린 데이터)와 뉴스 기반 실시간 데이터(빠른 데이터)를 비교하는 '이르미만의 선행 지표 그래프'를 넣어줘.

인사이트 리포트 카드: AI가 분석한 '오늘의 위기 요약'을 카드 형태로 정갈하게 배치해줘.

3. 구현 방식:

기존에 작성된 [여기 진주님의 기존 로직 코드나 변수명을 넣으세요] 로직을 그대로 활용해서 UI만 입혀줘.

사용자가 데이터를 클릭했을 때 애플 스타일의 부드러운 모달이 뜨게 해줘."
